🗄 Database Setup Guide

Follow these steps to connect your site to a real database so testimonies are shared across all visitors.

Step 1. Go to supabase.com → create a free account → New Project.

Step 2. In your project, go to SQL Editor and run this:

-- Testimonies table
create table testimonies (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  role text,
  body text not null,
  amens int default 0,
  approved boolean default true,
  created_at timestamptz default now()
);

-- Tags table
create table testimony_tags (
  id uuid primary key default gen_random_uuid(),
  testimony_id uuid references testimonies(id) on delete cascade,
  tag text not null
);

-- Comments table
create table testimony_comments (
  id uuid primary key default gen_random_uuid(),
  testimony_id uuid references testimonies(id) on delete cascade,
  author text not null,
  body text not null,
  created_at timestamptz default now()
);

-- Allow public read & insert (no login required)
alter table testimonies enable row level security;
alter table testimony_tags enable row level security;
alter table testimony_comments enable row level security;

create policy "public read" on testimonies for select using (approved = true);
create policy "public insert" on testimonies for insert with check (true);
create policy "public read tags" on testimony_tags for select using (true);
create policy "public insert tags" on testimony_tags for insert with check (true);
create policy "public read comments" on testimony_comments for select using (true);
create policy "public insert comments" on testimony_comments for insert with check (true);

Step 3. Go to Project Settings → API and copy your Project URL and anon public key.

Step 4. Open config.js in this site's folder and paste them in:

const SUPABASE_URL = 'https://your-project.supabase.co';
const SUPABASE_KEY = 'your-anon-key-here';

Step 5. Push to GitHub and enable GitHub Pages (Settings → Pages → Deploy from main branch).

✓ That's it! All testimonies will now be stored in Supabase and visible to everyone who visits your site.
